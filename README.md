# Text Adventure Game
A classic branching text adventure game built purely with Python standard libraries. Navigate the story by making choices directly in your terminal — no extra libraries, no complicated setup.

## Features
- Branching story with multiple choices and different outcomes
- 100% built with Python built-in functions (`print` & `input`), no third-party dependencies
- Runs on Windows, macOS and Linux
- Simple and intuitive text-based controls

## Requirements
Only **Python 3.x** is needed. You don't need to install any additional packages.

## How to Run
1. Download or clone this repository
2. Open your terminal / command prompt, go to the project folder
3. Run the game with:
   ```bash
   python main.py
